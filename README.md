# Ozon · Мебель — аналитика рынка за квартал

Подробный аналитический отчёт по рынку мебели на маркетплейсе Ozon: обороты, динамика роста, концентрация и монополизация ниш, лидеры и экономика продавцов. Подготовлен как основа для последующего сбора досье по компаниям (через Checko).

## 📊 Смотреть отчёт

**→ https://abdublinn.github.io/ozon-mebel-analytics/**

Одностраничный HTML: 9 разделов, 13 инфографик, интерактивные сортируемые таблицы, все цифры с пояснениями.

## Что внутри

- **Резюме** — ключевые выводы и цифры рынка (≈116 млрд ₽ за квартал, 17 категорий).
- **Обзор рынка** — все категории + карта рынка (treemap).
- **Динамика роста** — матрица «Рост × Размер», топ растущих ниш (без сезонного садового).
- **Концентрация и монополизация** — доли №1 / топ-2 / топ-5, индекс HHI.
- **Карта ниш** — размер × конкуренция × средний чек.
- **Игроки** — «многостаночники», портфели, приоритетный тир для досье (25 лидеров).
- **Экономика лидеров** — цена × объём, премиум vs рынок, доля выкупа.
- **Приложение** — полные таблицы (219 подкатегорий) + выгрузки.

## Данные

Папка [`data/`](data/): Excel (6 листов) + CSV (категории, подкатегории, концентрация, продавцы, метрики лидеров).

## Источник и оговорки

- Источник — рейтинги продавцов Ozon по категориям за квартал.
- Оборот = розничные продажи на маркетплейсе (с НДС), не финансовая отчётность юрлиц.
- Сезонная садовая мебель исключена из графиков динамики (учтена в объёмах).
- Данные — снимок за квартал; юрлица и цифры проверяются через Checko при сборе досье.
